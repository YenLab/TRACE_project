# This folder was adapted from IniMotif (reference: https://github.com/chengl7/IniMotif).
# It is used to generate motif matrices from HT-SELEX data.

### Steps:
1. **Clone the repository**:  
   Use the following command to clone the entire IniMotif repository:  
   
   git clone https://github.com/chengl7/IniMotif
   

2. **Modify the following files in the IniMotif-master directory**:  
    `inimotif_core.py`  
    `inmotif_main.py`

3. **Run the script**:  
   Execute the `inmotif_main.py` script in PyCharm or any other environment that supports a Python runtime.
